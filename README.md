# Binance Futures EMA RSI Scanner with Social Sentiment
A web-based crypto futures scanner combining technical analysis (SMA, RSI, candlestick patterns) with social sentiment data from Fear & Greed Index and CoinGecko.

## Features
- Top volume Binance USDT futures pairs scanning
- Backtesting with SMA-based strategies
- Live BUY/SELL signals
- Market sentiment analysis with Fear & Greed Index
- Per-coin social sentiment from CoinGecko
- CSV export with full trade details
